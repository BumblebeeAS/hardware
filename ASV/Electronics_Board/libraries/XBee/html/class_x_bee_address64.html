Detailed Description

Represents a 64-bit XBee Address

Note that avr-gcc as of 4.9 doesn't optimize uint64_t very well, so for the smallest and fastest code, use msb and lsb separately. See https://gcc.gnu.org/bugzilla/show_bug.cgi?id=66511
