Note that avr-\/gcc as of 4.\+9 doesn\textquotesingle{}t optimize uint64\+\_\+t very well, so for the smallest and fastest code, use msb and lsb separately. See \href{https://gcc.gnu.org/bugzilla/show_bug.cgi?id=66511}{\tt https\+://gcc.\+gnu.\+org/bugzilla/show\+\_\+bug.\+cgi?id=66511}
