88 // The following functions are intended to be used as callbacks, to
89 // print various information about received responses. All of the
90 // require a Print* to be passed as the data parameter. For example, to
91 // print to Serial any TxStatusResponses that contain errors, do:
92 //
93 // xbee.onTxStatusResponse(printErrorCb, (uintptr_t)(Print*)&Serial);
94 //
95 // Most of these callbacks can either be used as a response-specific
96 // callback, to only work that specific API response type, or as a
97 // generic callback (onResponse or onOtherResponse), in which case the
98 // relevant version of the callback will be called automatically.
