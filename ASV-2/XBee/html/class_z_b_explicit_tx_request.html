Detailed Description

Represents a Series 2 TX packet that corresponds to Api Id: ZB_EXPLICIT_TX_REQUEST

See the warning about maximum packet size for ZBTxRequest above, which probably also applies here as well.

Note that to distinguish reply packets from non-XBee devices, set AO=1 to enable reception of ZBExplicitRxResponse packets.

Creates a unicast ZBExplicitTxRequest with the ACK option and DEFAULT_FRAME_ID.

It uses the Maxstream profile (0xc105), both endpoints 232 and cluster 0x0011, resulting in the same packet as sent by a normal ZBTxRequest.

ZBExplicitTxRequest::ZBExplicitTxRequest ( )

Creates a default instance of this class. At a minimum you must specify a payload, payload length and a destination address before sending this request.

Furthermore, it uses the Maxstream profile (0xc105), both endpoints 232 and cluster 0x0011, resulting in the same packet as sent by a normal ZBExplicitTxRequest.
